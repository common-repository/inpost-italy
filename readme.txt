== Description ==

InPost per WooCommerce ha contribuito a migliorare l'integrazione e la collaborazione con le azienda che utilizzano Wordpress, facilitando la gestione delle spedizioni e delle etichette.

Quali sono i vantaggi dell'installazione del plugin?

* Accesso a tutti i servizi, descrizioni e branding di InPost;
* Mappa al Checkout aggiornata con Locker e Punti di Ritiro;
* Creazione Spedizioni e controllo Tracking;
* Generazione delle etichette di spedizione.

Dopo aver installato il plug-in, è necessario configurare la modalità di consegna InPost a seconda del servizio stipulato in fase contrattuale.

La documentazione completa e le istruzioni di installazione del plugin sono reperibili all'indirizzo:
[Documentazione InPost per WooCommerce](https://developers.inpost.it/#tag/Woocommerce)

Se hai domande o problemi relativi al plugin, contattaci tramite la mail
**integrations@inpost.it**

### Utilizzo di API di terze parti o servizi esterni

 

Per utilizzare il plugin, è necessaria una chiave API per l'ambiente di [produzione](https://api-shipx-it.easypack24.net) o di test [sandbox](https://stage-api-shipx-it.easypack24.net).

 

I servizi utilizzati all'interno del geowidget sono:

* [Google Maps](https://www.google.com/maps/dir/)

* [Openstreetmap](https://www.openstreetmap.org)

* [API of geowidget](https://geowidget.easypack24.net)
 

È possibile visualizzare i documenti [Politica sulla privacy](https://inpost.it/politica-sulla-riservatezza), [Termini e condizioni](https://inpost.it/sites/default/files/2022-08/20220901_Originali_InPost-T_C-ITA_vle-Cassala.pdf), [Compliance](https://inpost.it/compliance).

An API key in [production](https://api-shipx-it.easypack24.net) or [sanbox](https://stage-api-shipx-it.easypack24.net) mode is required for the plugin to work.
